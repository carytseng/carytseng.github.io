Mac平台下工具总结

常用使用mac平台总结出的优良体验，日常的效率工具，有助提升个人工作效率

必备工具

  • 解压：Bandizip
  • 影音：IINA
  • 终端替代：iTerm2
  • 文档阅览：PDF Expert
  • 浏览器下载：NeatDownloadManager
  • 磁力链接必备：迅雷
  • 系统清理：Disk diet
  • 本地索引搜索：HapiGo
  • 外网工具：Clash
  • 文档编辑：wps
  • 邮件收发（主要可设代理，方便gmail）：Formail
  • 文件管理器替代：TotalFinder

工作必备

  • MarkDown编辑：Typora
  • 文本编辑：Sublime Text
  • Java开发：IDEA
  • 数据库图形化：Navicat
  • git图形化工具：SourceTree
  • svn工具：Cornerstone
  • 接口调试：PostMan
  • UML工具：draw.io
  • 容器：docker
  • 性能监控：JProfiler
  • 文本对比：Beyond Compare
  • redis图形化工具：RDM
  • 前端编辑：Visual Studio Code
  • ssh工具：Royal Tsx
