JWT

2020-06-26
JSON Web Token

定义

是一种跨域认证的令牌技术。

解决场景

  1. 用户向服务器发送用户名和密码。

  2. 服务器验证通过后，则在session中保存用户的相关信息，如用户角色，权限等。

  3. 服务器返回一个sessionId给客户端 ，写入浏览器的cookie中。
