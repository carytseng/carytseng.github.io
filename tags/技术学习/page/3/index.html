count的用法

*count()、count(id)、count(1)、count(字段)的区别，针对MySQL**

count(id)，InnoDB会遍历整张表，把每行的id取出来，返回给server层，判断不为空，按行累计。

count(1)，InnoDB会遍历整张表，每行放数字1进去，返回给server层，判断不为空，按行累计。
